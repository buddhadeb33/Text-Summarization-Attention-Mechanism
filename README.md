# Text Summarization
_Sequenece to Sequence Modelling_

**Attention Mechanism**

![](images/banner.jpeg)


This is a NLP project for Text Summarization which is built with Flask(RESTapi) and deployed on Heroku(PaaS) using NLTK for summarizing text applied attention mechanism.
